## Built With

* **HTML5 & CSS3** — Native semantic structure with custom responsive dark theme styling.

* **Tailwind CSS** — Utility-first styling framework used for modern, premium design aesthetics.

* **GitHub** — Repository hosting and version control.

* **Vercel** — Automated CI/CD deployment.

## Updating the Portfolio

This repository is connected to Vercel via a continuous deployment pipeline. Any changes made to the `index.html` file will automatically trigger a new build, updating the live production website within seconds.
